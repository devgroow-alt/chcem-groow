# chcem-groow

Static Next.js clone of the GROOW Digital landing page at chcem.groow.sk.

## Tech Stack
- Next.js 15 with App Router
- TypeScript
- Tailwind CSS v4
- Static export (output: "export")

## Architecture
- All components in `src/components/`
- Single page at `src/app/page.tsx`
- Local images in `public/images/`
- Fonts: Big Shoulders Display + Questrial via next/font/google
